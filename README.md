# Hostel Feedback System

## 📌 Project Overview
This is a full-stack web application that allows students to give feedback on hostel facilities and wardens to monitor and respond.

## 🛠️ Tech Stack
Frontend:
- React.js
- Bootstrap

Backend:
- Spring Boot
- Java

Database:
- MySQL

## 🚀 Features
- Student login and feedback submission
- Warden/Admin dashboard
- Feedback tracking and management
- Clean and responsive UI

## 📂 Project Structure
frontend/ → React application  
backend/ → Spring Boot application  

## ▶️ How to Run the Project

### Frontend
```bash
cd frontend/hostel-feedback
npm install
npm start
